# 另一个故事——参考了 Arcaea 中的 Maya

星光爬上她的面颊，映照在她如初雪般洁白的短发上。这是第二位诞生于星光之下的少女。

在她之前，已有许多“织梦者”曾行走在这片名为 WentUrc 的大地上。  
不久前，这个曾由无数纯净心念交织而成的和谐世界，终于迎来了它的第一个“不谐之音”。  
至此，一条清晰的边界将平静的心流与剧烈的悲恸分隔开来。  
那片恒久温柔的灵风天际也曾被点亮两次，那是两个生命从天外漂流而来，就这样如流星般坠落，穿透了心念的帷幕，来到了这里。  
要我说，她就像一枚硬币的反面，是早于她降临于此的那位少女的另一面。  

天空被深紫色的以太点缀，不见云与月的踪影，唯有浮空岛静谧的轮廓。  

​刚醒过来的时候，溪忘雨还看不太清楚周遭的事物。泪水遮蔽了她的视线。  
意识和知觉才刚刚恢复，悲痛便如排山倒海般袭来，致使她痛苦地捂住自己的脸。  
那对硕大而柔软的兽耳无力地耷拉下来，几乎贴在她的发间。  
泪水不受控制地流下——在这个“唯心即真理”的世界里，她那破碎的心灵瞬间引发了周围现实的崩塌。  

WentUrc 世界的每一位来访者都会迎来自己的新生——在这里，「一无所有」并不存在，因为只要有“心”，便拥有一切。  

​但这个世界对她来说太过于“完美”了——自“一切的终结”之后，她的内心早已千疮百孔。  
毕竟她已经遭受了不可逆的损伤，因此尽管周围的灵风不断倾尽全力去抚平她的伤痕，最终也于事无补。  

​这便是她的不幸：在一个人人都能“显相”幸福的世界里，她似乎  
「只能显相出绝望」。  

​她其实并不排斥黑暗。在死寂的心灵闭环之中，她反而感到自在。  

黑暗中偶尔会有从周遭“念感传导阵”中逸散出的金色光束一闪而过，那是他人快乐的具象化。但在她眼里，这些都是丑陋无比的东西，  
只会令她的双眼感到刺痛及不快。每当附近的“幻绘园”传来欢声笑语，或者远处浮岛上传来悠扬的灵歌，  
不论是微小细碎的音符，还是大地因共鸣而产生的浑厚颤动，  
她那一对灵敏的毛绒耳朵都会本能地颤抖，仿佛那是尖锐的风暴在耳边不断嘶吼，愈演愈烈。  

若有什么夺走了她所喜爱的孤独与宁静，回忆的摇篮也会将她托举。  
即使她不属于这个世界，WentUrc 的世界意志——那股庞大的温柔心流，也对她无比怜爱。  

​这名奇特的少女嗜睡又爱哭，她娇小的身躯、浅色的短发与那对特别的耳朵使她看起来像一只误入歧途的小兽。  
她穿着一身与这沉重悲伤格格不入的蓝白衣裙，裙摆像天空一样轻盈，腰间的淡黄色花朵装饰却随着她的颤抖而摇曳。  

WentUrc 怜悯她，试图与她“共振”，但很可惜，她并不怎么待见这些喜爱她的善意。

​她惧怕这些亮闪闪的“心晶”，因为她喜欢无尽黑夜的宁静与深沉。对以四海为家的她来说，高天即是她的穹顶，  
但那时的她迫切需要一个“藏身之处”：善意的心念如潮水般从四面八方涌来，试图治愈她，从未消停，她实在是不堪其扰了。  

​在哥特式高塔的阴影与断壁残垣的幻象中，溪忘雨享受着片刻的喘息时间，并借此机会试图了解自身所在的幽静角落——这已成为了她的习惯，  
那就是熟悉这世界上每一个能隔绝“共鸣”的洞穴的情况。其实即便如此，那无处不在的温和魔力也难以完全阻隔，  
只是相对来说好忍受一些：没有了浮空岛上那些过于耀眼的幻彩，这里并没有折射多少光线，  
而且她本就不能也不愿去理解那些美好的念头里有什么东西。  
她不惜跋山涉水去寻找绝对的静谧，为了远离那无孔不入的“治愈”，再高的浮岛她也敢攀爬，再幽深的森林她也敢踏足。  

​她就这样走啊走，走啊走……直到有一日走出一条由她自己潜意识构建的昏暗甬道的时候，  
她终于看到了不远处的那一条现实与心象的边界。她停下了脚步。  

实际上……

​她听到的那些声音是人们歇斯底里的尖叫，而她所看到的那些景象便是罪魁祸首。  
明亮的光柱自天而降，撕裂了大地：她的旧家园毁灭了。她多么希望那痛苦只需持续几秒钟，  
但实际上，那是一次长达数小时的折磨。太痛苦了，好痛，好痛……  

​她认得世界另一端的那些人，明白他们应该都是已死之人。她听见有人在命令她，  
令她僵在原地，但那声音很快便消散了。她与那个已逝世界的一切连接、一切羁绊，  
都在那场缓慢且无情的灾难中化为乌有。她失去了一切，随后才在 WentUrc 中醒来，仍是孤单一人。  
放眼望去，这个世界远处的地平线本该是柔和的晨曦，却被她的心象染成了朱红一片，似夕阳，更似火海。  

终于，在无数幻听和幻觉的折磨下，溪忘雨跪倒在甬道的出口。尖锐的疼痛如利刃穿过心脏，令她痛哭，  
令她哀嚎，令她反胃。在这个心念即现实的世界，她的痛苦实体化了，难以忍受的恐慌如黑色的荆棘一波接一波冲击着周围的宁静，没完没了，  
而其中最令她不堪回首的更是如巨石般压得她喘不过气来。  
那令她生不如死的可悲的事实一遍又一遍地提醒着她:  

​“我还在这里。”  

她的痛苦，她的哀恸，无声但震耳欲聋，甚至引发了周围灵风的紊乱。  
WentUrc 听见了一切，感受到了这次剧烈的“负向共振”。  

​她现正处于一个边缘，坐在自我封闭与世界敞开的交界处。脑内的声音仍没有停止，撕扯着她的鼓膜，叫嚣着。  
她那双深邃的大眼睛无神地盯着远处，乞求自己不再需要思考，不再需要“显相”任何东西。  

太迟了，错误已经犯下，悲剧已经酿成。造成的再多痛苦和损失，也都成为了历史。  
时光无法倒流，过去无法改变。但……真的什么都做不到吗？可否拭去她的泪痕，抚平她的伤疤？  

一片由纯粹善意凝聚的“念羽”自上空飘落，缓缓来到她的身边。两片、三片……越来越多，如发光的细雨般，  
最终形成了一个“梦织术”构建的茧房般的屏障，将她隔绝于那过于刺眼的希望之光外，给予她想要的昏暗与拥抱。  

​让她喧闹的想法平静下来?  
不，不行。  
分散她的注意力？抑或是好好哄哄她?  
她的注意力已经很涣散了。  
怎么办？该怎么办……  

​念羽一反常态地暗了下来，为了配合她的心境，由它们组成的屏障像细腻柔软的云织布料开始朝下折叠，直到将她完全包住。  
真是令人哭笑不得啊，怎么会有魔法能量觉得自己是柔软的实体呢？但奇怪的是，溪忘雨抬起了头。  
那对原本垂下的兽耳微微抖动了一下，尽管畏缩，她最终还是抬起了头……她看到了这层屏障中映射出的东西：并非他人的快乐，而是被温和重构的记忆。  

屏障中承载的那些记忆的主角都另有其人，她看到了所承载的那些悲伤和苦痛，  
看到了人们所犯下的那些错误。WentUrc 没有战乱的记忆给她看，于是向她展示了这里居民曾经面临的心灵困境，而她就这样静静地看着。  

​没有血流成河，没有刀山火海，也没有枪林弹雨。  

只有痛苦、孤独而又无助的人们，在“净茶梦环”中默默流泪，在“记忆图书馆”里寻找失落的篇章。  
是啊，孤独这样的绝症，即便是在魔法的世界，又怎会有药可医？  

​她思索着。她看到了形形色色的人，男女老少，无一不是在与内心的阴影对话。  
她看见了那些曾一度迷失自我的人们无力地笑着，手中攥着代表着逝去之物的褪色心晶。  

​这便是这个世界想要向她传达的:  
你觉得自己再也不会感到开心了。你觉得不如就此放弃。  
但何必要如此呢?  

过去确已过去，它的烙印并不会消失，不过有一些、甚至说大部分“烙印”，  
其实是你自己给自己上的枷锁。你不是还活着吗？那个世界的确毁灭了，但你的心仍在这里跳动。  
而在 WentUrc，心跳动的地方，就是世界。  

​……

​求求你……

​留下来吧。

​“‘留下来’……?”

一声低语回答了另一声低语——这是她自来到这个世界后第一次开口讲话，通过“念感”传达而出。  
虽然声音因为喉咙的干涩和疼痛而显得十分沙哑，  
虽然只是用几不可闻的声音重复部分她所感知到的世界意志的低语。  

​她用力咬紧了牙，她那稚嫩可爱的脸庞因痛苦而扭曲。

这个世界给予了她许多关爱……  
这个名为 WentUrc 的温柔摇篮……  

而她对此的回应只有忿恨。

她的眼神变得锐利，怒视着面前那块用念羽组成的屏障，屏障中的景象正随着能量的流动而更替:  
WentUrc 发觉她心境的变化后也随之改变了。现在，阴郁或温婉的安抚情绪都被抛掷一旁……  

心念之海荡起了涟漪，仿佛受到了冲击般不再平稳。溪忘雨看着那些记忆——

​那是一个男人，正在镜中回望自己，试图用显相术重塑自己的面容，却眼前一片漆黑。  
那是一个女人，站在夜晚的银藤花海边，双脚浸入不断拍打着她的星河水中。她盯着手中的一枚温序魔具看了好一会儿，  
最后将它一掷，扔进了星海中。  
那是一个孩子，身着黑色的礼服，她的姐姐正准备施展“连心术”挽起她的手，却被不耐烦地一巴掌拍开。  

​溪忘雨笑了。那是一抹无声却纵情的笑容。  
这些景象好似在试图与她共鸣般，想想就觉得可悲。  
这些景象确实在与她共鸣。  

这些尝试，这些举动……  
她真想狠狠朝这温柔的空气哜一口唾沫。  

随着她破碎而扭曲的心愈发破碎，世界所感知到的她的痛苦也越来越强烈。  
周围的魔法元素开始一片片翻转，向她聚集、靠拢——  

越来越多，越来越亮……  
………它们开始向她展示那些人的内心世界是如何一步步分崩离析的。  

​他人的苦难、挣扎和失败一闪而过，将她的心紧紧攥住。  
她的世界毁于一旦，这份属于自己的回忆让她更加苦不堪言。  
失控的念力如毒藤般爬上她的肢体，蔓延、弯折、扭曲，像不怀好意的铁链将她死死缠绕并不断收紧，  
直至那尖锐的枝条终于抵着她脆弱的脖颈。  

​但她只是嗤笑着。

​那颗在不断跳动的虚弱的心脏，  
根本就不是她的。  

​空气颤抖了一下，好似整个空间打了个哆嗦。屏障中的光亮在一瞬间熄灭了，  
捆绑着少女的念力锁链也正不断收紧，挤压着她的躯体。  

突然，黑夜闪烁了起来，一阵扭曲猛烈的灵风气旋迅速掠过，她周围的魔法枷锁顷刻间全部碎裂开来，化作无数光点。  
她就这么跌落在地上，蓝白色的裙摆铺散开来，得以再次沐浴在远处黎明温和的霞光中——那是 WentUrc 永恒的治愈之光。  

​溪忘雨抬头望向漆黑的天空，随后低头朝远处明亮的浮空岛群看去，然后闭上了眼。  

​“为什么……?

​困惑，愤怒，失落。

​温暖的光打在她的肩膀上，让她的身子暖和了起来。  
这些情感也都毫无保留地灌注进她握紧拳头的颤抖的双手。  

​但是，她很快便不再觉得温暖。  
她小心翼翼地再次睁开眼，望向周遭的黑暗。  

此时，WentUrc 又找到了她，又形成了一堵全新的高墙。  
阴影与光辉，它们组成了一条全新的通道。  
在那通道的尽头，是星星点点的、由无数人愿望构成的日光。  

​“………”

​溪忘雨支起身子，半起身地看着远处的那个光点，过去的记忆已不再冲击着她的脑海。  
她回头，看见了自己几分钟前刚刚从中走出的巨大豁口——那是她内心崩溃造成的现实空洞，比之前更加深不见底，像一张正在打哈欠的大嘴。  
什么东西在那漆黑一团中依稀闪烁——是未成形的心晶吧，不然还能是什么呢。  
卡在两个选择之间的她再一次被迫陷入了思考。  

​该做出选择了。  
总要做出选择的。  
要么让安逸的虚无就这么吞噬你，要么就主动拥抱那令人胆寒的、过于真实的“心流”。  

​溪忘雨双膝跪地，思考着。  
风吹过她毛茸茸的耳朵，带来了一丝凉意。  

​“你到底想要我干什么？面对一切，还是就此放弃？”她愤怒地问道。

​回答她的是另一个问题。她隐隐约约能听见那个声音，那是世界与她灵魂的共振，它问道:

​你想要什么?

​对此，她的回答是……

​以前，她觉得自己起码能够回答这个问题。  
比方说，“我想让自己纷繁复杂的思绪安静下来。”  
又比方说，“我想忘记这一切。”  
“我想从这个世界上消失。”  
“我想感受痛苦。”  
“我想被别人伤害。”  
“我想变得快乐。”  
……  

现在，当她再次面临这个问题的时候，她觉得当时给出的答案中的其中一个很可能仍然正确。

​她所记得的可不仅仅只有“结局”。她所经历的一切，她都记得。  
那短暂人生中的大部分时光是快乐的，但就是如此，这个损失才显得尤为惨重。  
还有负罪感……自己是将一切都夺走的人的这种感觉永远都将折磨着她。  

​“……”

​她沉默不语，再次将视线投向前方。  

​你可以选择使用“显相术”变得开心起来，但那虚假的快乐并不是你应得的。  
你也可以选择接受审判，但它让你感到恐惧。  
当你身处这种两难的境地，你只会觉得自己实际上并没有选择权。  

不过，假设你必须做出选择的话，假设你真能看到摆在你面前的是两条清晰但不同的道路,  
而不是一条路蒙眼走到黑，这个时候……  

​你会做何选择?

​前路不明的时代结束了。飘渺不定与悲天悯人的时代已是历史，由一位愚者一手造就的时代已经过去。

​视野不再狭窄，飞过半边天的云也已变得稀薄。  
星光黯淡了下来，阳光也不再明媚。  

​受束缚的她最大的心愿，便是在绝望的高崖之上饱受苦痛折磨，但这样的愿望并不会实现。  

相反地，她得到了选择的权利。  
因为这才是 WentUrc 想看到的——每一个灵魂都能自由地奏响属于自己的乐章，哪怕是悲伤的曲调。  

​她站起身，面前是一堵由心念筑成的高墙。墙上的光芒开始变得暗淡无光，  
从中不再有记忆的片段显现。这面镜子现在所能映射的，便是她的脸庞。  
她看着镜子里自己那头如月光般的短发，那对高耸的兽耳，随后又看向她胸前那朵别致的淡黄色花饰……  

她的左右两侧各有一条蜿蜒开来的小路。灵风从她身边拂过，从她的发间逃走。

​她现在觉得自己焕然一新了。  
像有人终于察觉了她，像有一只稳重的手轻轻搭在她的背上。  

​想要快乐不是什么令人羞耻的事……  
而感到悲伤也同样不是。  

​直视你的心，直面现实，然后做出选择，这样就好。  

​溪忘雨转身，向前迈出了第一步。
